* T cloud 샘플 코드를 다운로드해 주셔서 감사합니다.
* 이 코드를 실행하시기 위해서는 https://developer.skplanetx.com에서 
  Application Key 등을 발급 받으셔야 합니다.
* 발급 받으신 key를 assets/skpop.properties.sample에 붙여 넣고,
  해당 파일을 skpop.properties로 변경합니다.
* 계속적인 기능 개선을 위해서 libs 디렉토리에 포함된 SKPOP_SDK_*.jar 파일은
  추가적으로 업데이트 될 수 있습니다. 
  https://developers.skplanetx.com/develop/doc/sdk/에서 다운로드 받을 수 있습니다.
* 본 코드는 T cloud를 이용한 Application을 개발하기위한 Sample 코드이며,
  Application 작성에 필요한 참고용으로 사용해주십시오.
* T cloud의 회원 가입은 http://www.tcloud.co.kr에서 가능합니다.
